# Corda Enterprise

Corda Enterprise is R3's closed source patch set on top of Corda Open Source. It adds features and improvements that we
plan to charge for.

## Extra features

* Doorman
* SOCKS relaying
* Flow triage screen in Explorer
* No stupid jokes at startup
* SGX
